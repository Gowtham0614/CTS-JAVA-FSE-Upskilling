Exercise 5.2 – Column Reordering with order-md-*

On mobile: HTML order (A, B, C). On md+: reordered (B, C, A) using order classes.

A — HTML First
order-md-2
B — HTML Second
order-md-1
C — HTML Third
order-md-3

On desktop, column B appears first (order 1), A appears second (order 2).

<div class="col-md-4 order-md-2">A (appears 2nd on desktop)</div>
<div class="col-md-4 order-md-1">B (appears 1st on desktop)</div>
<div class="col-md-4 order-md-3">C (appears 3rd on desktop)</div>
