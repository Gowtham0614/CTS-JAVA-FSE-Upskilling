19.1 Set Up Bootstrap 5 with Sass using npm
19.1
Project Setup

Bootstrap's Sass source files let you customize every design token before compilation — colors, spacing, typography, and more — instead of overriding compiled CSS after the fact.

1
Create a folder and run npm init -y to generate package.json
2
Install packages: npm install bootstrap@5 sass
3
Create scss/custom.scss — your Sass entry point
4
Add compile script to package.json and run npm run sass
5
Link the output css/custom.css in your HTML
📦 package.json — npm scripts
// package.json
{
  "name": "bootstrap-sass-project",
  "scripts": {
    // one-time compile
    "sass": "sass scss/custom.scss css/custom.css --style=compressed",
    // watch mode (auto-recompile on save)
    "sass:watch": "sass scss/custom.scss css/custom.css --watch"
  },
  "dependencies": {
    "bootstrap": "^5.3.3",
    "sass": "^1.77.0"
  }
}
📄 scss/custom.scss — entry point
// 1. Your overrides (BEFORE Bootstrap)
@import 'variables';

// 2. Bootstrap required files
@import '../node_modules/bootstrap/scss/functions';
@import '../node_modules/bootstrap/scss/variables';
@import '../node_modules/bootstrap/scss/variables-dark';

// 3. Maps, mixins, utilities
@import '../node_modules/bootstrap/scss/maps';
@import '../node_modules/bootstrap/scss/mixins';
@import '../node_modules/bootstrap/scss/utilities';

// 4. Bootstrap layout & components
@import '../node_modules/bootstrap/scss/reboot';
@import '../node_modules/bootstrap/scss/grid';
@import '../node_modules/bootstrap/scss/buttons';
// ... add only what you need
@import '../node_modules/bootstrap/scss/utilities/api';

// 5. Your own custom styles below

19.2 Customize via _variables.scss & Recompile
19.2
scss/_variables.scss

Bootstrap variables use !default — they only apply if not already defined. Import your overrides before Bootstrap and your values win. 

// ─── Color Overrides ───────────────────
$primary:    #e63946;   // was #0d6efd
$secondary: #457b9d;
$success:   #2a9d8f;
$info:      #e9c46a;
$warning:   #f4a261;
$danger:    #c1121f;
$light:     #f1faee;
$dark:      #1d3557;

// ─── Border Radius Overrides ───────────
$border-radius:      0.75rem;  // was 0.375rem
$border-radius-sm:   0.4rem;
$border-radius-lg:   1.25rem;
$border-radius-pill: 50rem;

// ─── Typography ────────────────────────
$font-family-sans-serif: 'DM Sans', system-ui;
$headings-font-weight:   700;

💡 Key rule: The !default flag in Bootstrap's variables means "use this value only if nothing else has set it." Import your file first → Bootstrap's defaults are skipped.
