Exercise 1: Setting Up Bootstrap 5

Exercise 1.1 – CDN Setup

This page links Bootstrap 5 via CDN in the <head>.

✅ Bootstrap CSS loaded via CDN: cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css
✅ Bootstrap JS bundle loaded at bottom of <body>: bootstrap.bundle.min.js

Exercise 1.2 – npm / Downloaded Files (Notes)

To set up via npm:

npm init -y
npm install bootstrap@5.3.3

Then reference in your HTML:

<link rel="stylesheet" href="node_modules/bootstrap/dist/css/bootstrap.min.css">
<script src="node_modules/bootstrap/dist/js/bootstrap.bundle.min.js"></script>
📁 Alternatively, download Bootstrap from getbootstrap.com and reference the local files in the same way above.

Bootstrap version: 5.3.3
