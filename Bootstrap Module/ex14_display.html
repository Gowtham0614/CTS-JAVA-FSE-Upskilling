Exercise 14: Display & Visibility

Exercise 14.1 – d-none, d-md-block, d-lg-flex

Resize the browser window to see elements appear/disappear at different breakpoints.

Syntax: d-{value} or d-{breakpoint}-{value}
Values: none, block, inline, inline-block, flex, inline-flex, grid, table
📱 MOBILE ONLY (d-block d-md-none) — hidden on md and above
🖥️ TABLET & DESKTOP (d-none d-md-block) — hidden on mobile
💻 DESKTOP FLEX (d-none d-lg-flex)
📊 SM: inline-block → MD: block (d-none d-sm-inline-block d-md-block)
🌐 ALWAYS VISIBLE (no d- class)
