4 DevTools Quick Reference
  • 🖱️ Inspect Element: Right-click any element → Inspect, or press F12 and click the cursor icon. Edit CSS in the Styles pane in real time.
  • 📋 Console: Open DevTools → Console tab. Use console.log(), console.warn(), console.error(), console.table() in your scripts.
  • 🔴 Breakpoints: Go to DevTools → Sources → open your JS file → click a line number to set a breakpoint. The page pauses at that line. Inspect scope variables in the right panel.
  • 🔬 debugger statement: Write debugger; in your JS. When DevTools is open, execution pauses at that line automatically — no clicking needed.
  • 📱 Device Mode: Press Ctrl+Shift+M to toggle the device toolbar and test responsive layouts at different screen sizes.
  • 🌐 Network Tab: See all HTTP requests, response codes, and payload sizes. Useful for debugging API calls like fetch() or form submissions.
