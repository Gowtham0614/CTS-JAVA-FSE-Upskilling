visibility: hidden  vs  display: none

Box A – Visible ✓
Box B – visibility: hidden
Box C – Visible ✓
Box D – display: none
Box E – Visible ✓
visibility: hidden – the element is invisible on screen, but it still occupies its space in the layout (you can see the gap where Box B was).

display: none – the element is completely removed from the page. No space is reserved. Other elements flow as if Box D never existed.
